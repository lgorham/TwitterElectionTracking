About

The data in this project was scraped and parsed from Twitter's live search webpage using Selenium WebDriver and Beautiful Soup. Due to limitations in the data scraping process, the data is limited to a sampling of tweets from each day and should not be used for research purposes.

The sentiment analysis is conducted using a Bernoulli Naive Bayes classifier trained on a set of hand-tagged training corpus of 500 tweets, classified as either negative or positive
